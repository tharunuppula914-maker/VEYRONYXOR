# TaskFlow - Mobile Productivity & Team Collaboration Platform

## Project Overview

TaskFlow is a modern, production-ready mobile application built with React Native that combines task management, calendar, reminders, notes, and team collaboration features.

**Target Users:** Students, job seekers, freelancers, professionals, startups, and small teams.

## 🎯 Key Features

### Phase 1 - MVP
- ✅ Authentication (Email, Google Sign-In)
- ✅ Task Management (CRUD operations)
- ✅ Kanban Board (Drag & Drop)
- ✅ Calendar (Multiple views)
- ✅ Reminders & Notifications
- ✅ Notes (Rich text)
- ✅ Analytics & Dashboard
- ✅ Focus Timer (Pomodoro)
- ✅ Settings & Profile

### Phase 2 - Team Collaboration
- 🔜 Workspace System
- 🔜 Team Management
- 🔜 Task Assignment
- 🔜 Activity Timeline
- 🔜 File Attachments

### Phase 3 - Advanced Features
- 🔜 Direct Messaging
- 🔜 Meeting Scheduler
- 🔜 AI Assistant
- 🔜 Attendance Tracking

## 📱 Tech Stack

**Frontend:**

## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- Expo CLI or React Native CLI
- Android Studio / Xcode

### Installation

```bash
# Clone repository
git clone https://github.com/tharunuppula914-maker/VEYRONYXOR.git
cd VEYRONYXOR

# Install dependencies
npm install

# Start development
npm start
```
